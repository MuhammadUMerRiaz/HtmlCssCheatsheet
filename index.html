Html Table tags

Tag Name Description
<table> Table The wrapper element for all HTML table
<thead> Table Head The set of row defining the column headers in a table
<tbody> Table Body The set of row containing the actual data in a table
<tr> Table Row The table row container
<td> Table Description The table row container
<tfoot> Table Footer The set of row defining the footer in the table

Css

selector Description
* for all element
div for div tags
div,p for all divs and paragraph
div p for all paragraph inside divs
div>p for all p one level deep in div
div + p p tags immediately after div
div ~ p p tags preceded by div
.classname all element with the class
#idname element with ID
div.classname divs with certain classname
div#idname div with certain ID
#idname* all the elements inside the specific id

Relative path

path with filename Description
picture.jpg The "picture.jpg" is located in the same folder as the current page
images/picture.jpg The "picture.jpg" is located in the images folder as the current page
/images/picture.jpg The "picture.jpg" is located in the images folder at the root of current page
../picture.jpg The "picture.jpg" is located one level up from the current page
